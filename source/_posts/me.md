### 🤡 以前 vs 现在

以前在公司，我的 Title 是 **"Senior Front-End Engineer"**（高级前端工程师）。
也就是俗称的：“那个写 Vue 的”。

现在，我自己出来单干了，我给自己印的名片是 **"Indie Developer"**（独立开发者）。
听起来很酷对吧？但实际上，我的真实职位描述包括但不限于：

* **CEO**：负责做梦和焦虑。
* **CTO**：负责把 CEO 的梦砍掉一半，因为做不出来。
* **客服专员**：负责跟客户解释为什么“把这个Logo放大的同时缩小一点”是逻辑悖论。
* **财务**：负责计算今天的午饭是不是超支了 2 块钱。
* **运维**：负责对着黑漆漆的命令行发呆。

### 🛠️ 重新定义的“技术栈”

以前面试，别人问我技术栈，我会自信地背诵：
> "精通 Vue3/React 源码，熟悉 Webpack 性能优化，了解 Node.js 微服务架构，熟练使用 Docker 容器化部署..."

现在，如果你问我做闲鱼单子和独立开发主要用什么技术？
我的**真实技术栈**变成了：

1.  **Ctrl+C / Ctrl+V 工程学**：面向 GitHub 编程，面向 Google 调试。
2.  **HTML (How To Make Low-budget-sites)**：客户不在乎你用什么框架，他们只在乎“能不能便宜点”。
3.  **Photoshop (盗版)**：没错，作为前端，我现在被迫学会了抠图、去水印、改文字。
4.  **心理学**：主要用于分析客户说“我就改这最后一点点”时，这句话的可信度通常低于 1%。

### 🐛 Bug 的降维打击

以前我处理的 Bug：
* *"在高并发场景下，这个 WebSocket 连接可能会导致内存泄漏..."* (听起来就很高端)

现在我处理的 Bug：
* *"老板，为什么我上传的身份证照片被压扁了？"*
* *"那个...我的网站怎么在 IE6 上打不开？"* (大清都亡了啊亲！)
* *"能不能把这个红色的按钮改成‘五彩斑斓的黑’？"*

### ☕ 自由的代价

大家总觉得“不上班”很爽，不用早起挤地铁，不用听老板画饼。
确实，我现在拥有了**绝对的自由**。

* 我可以在凌晨 3 点写代码。（因为白天在睡觉）
* 我可以随时来一场说走就走的旅行。（从卧室走到客厅）
* 我可以自己决定产品的方向。（虽然最后往往取决于客户给多少钱）

但说实话，这种**“手停口停”**的危机感，比任何闹钟都管用。它逼着我走出了“只关注技术”的象牙塔，开始去学习产品、学习运营、学习如何与人打交道。

### 🌟 写在最后

虽然整篇都在吐槽，但我并不后悔现在的选择。

在大厂，我是 huge machine 里的一颗精密的螺丝钉，由于打磨得太好，以至于我忘了自己其实可以是一把锤子、一把锯子，或者一根搅屎棍（划掉）。

现在，虽然每天都在“打杂”，但我真切地感觉自己**掌握了生活的方向盘**。哪怕车开得有点颠，哪怕偶尔会抛锚，但——这车是我自己在开啊！
